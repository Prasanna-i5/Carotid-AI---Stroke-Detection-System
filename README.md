# 🧠 CAROTID AI – Stroke Risk Prediction using AI

CAROTID AI is an AI-powered web application designed to assist in the **early prediction of stroke risk** by analyzing carotid artery CT images using **Machine Learning** and **Deep Learning** techniques.

The application enables users to upload CT images, perform automated analysis, and receive stroke risk predictions through an intuitive and responsive web interface.

---

## 🚀 Features

- 📤 Upload carotid artery CT images
- 🖼️ Image preprocessing and enhancement
- 🔍 GLCM feature extraction
- 🤖 Stroke risk prediction using **Support Vector Machine (SVM)**
- 🧠 Deep Learning prediction using **VGG19 CNN**
- 📊 Confidence score generation
- 📈 AI-assisted stroke risk assessment
- 💻 Responsive and user-friendly medical-themed interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/carotid-ai.git
```

Navigate to the project directory

```bash
cd carotid-ai
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
